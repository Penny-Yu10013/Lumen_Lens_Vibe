## 階段規劃（切片優先，3D 壓軸）

### P0 — 光桌切片模式（核心 MVP）
切片 = 薄圓盤/自由形網格 + 剖面圖案函數 + 背光透射。全部 2D 圖案數學，成本最低、視覺回報最高。

| 現象 | 圖案函數 | 光學要點 |
|---|---|---|
| 達碧茲 Trapiche | 六重角向扇區：角度取模 + 臂寬/臂濃度參數，扇區邊界富集暗色包裹體 | 透射光下臂是吸收體（暗），本體透光 |
| 李迪克碧璽 Liddicoatite | 沿 c 軸三角分帶：三重對稱距離場 + 同心色帶堆疊（粉/綠/褐色序可調） | 靶心三角，色帶邊界銳利度參數 |
| 彩虹瑪瑙 Iris Agate | 細密瑪瑙紋帶（噪聲扭曲的同心帶）+ **薄膜/光柵繞射**：帶間距 < 光波長級 → 視角相關的光譜色 | 只在背光+特定角度出彩虹 = 天然的互動獎勵機制，轉起來才有彩虹 |

- **光桌互動（核心體驗）**：切片可旋轉/傾斜，背光強度滑桿，環境反射開關。彩虹瑪瑙的彩虹隨傾角掃過 = 本作的第一個「哇」時刻。
- 每片附極簡標本卡（名稱/晶系/現象一行解釋，中英）。

### P1 — 星石手電筒模式
- 星光效應（asterism）：定向金紅石針 → 各向異性高光。實作 = tangent-based anisotropic specular，三組針軸 → 六射星。
- **手電筒 = 可拖曳點光源**，星芒跟著光走（現實中星石就是這樣玩的，互動即科普）。
- 星光紅寶/藍寶/星光粉晶三色可選。全黑背景，只有手電筒光斑。

### P2 — GIA 八面體星芒雲霧（3D 壓軸）
- 完整 raymarching：背面深度 pass → 入射/出射點之間 march 扇區函數（<111> 生長扇區距離場）。
- 氫雲霧密度參數、視差萬花筒效果（視角轉動時內部圖案位移）。
- UV 燈開關：扇區螢光 emissive（物理有據：生長區微量元素差異→螢光分區）。
- 效能紅線：march 步數手機 ≤48、桌機 ≤96；P2 獨立頁面載入，不拖累 P0/P1。

## 視覺與 UI
- **背景：純黑/近黑**（#000–#0a0a0f），寶石是唯一光源敘事。
- **UI：蘋果網美玻璃質感**——frosted glass 面板（backdrop-filter: blur + 半透明 + 細邊框高光），控制項懸浮在寶石旁。
- **流量密碼機制：寶石折射穿過 UI**——玻璃面板背後的寶石光暈/彩虹被 blur 折過去，UI 本身成為光學展示的一部分。實作：UI 玻璃用 CSS backdrop-filter（便宜），寶石光暈刻意溢出到面板底下。
- Logo：LL 疊字置於左上，發光細線風格，與黑底玻璃 UI 同語言。
- 字體與 Gemcraft 區隔：Gemcraft 是異世界工坊感，本作走實驗室/精品感（GIA 標本卡 × Apple 發布會）。
- 雙語 i18n 沿用前作方案。

## 成功判準
- P0 上線時：彩虹瑪瑙轉動出彩虹的 15 秒錄屏能直接當推特素材
- 深度層驗證：有人在回覆裡認出達碧茲/Liddicoatite 並開始科普 = 釣到目標魚
- 技術驗證：手機 60fps（P0/P1），P2 桌機 60fps / 手機 30fps 可接受

## 風險
- 薄膜繞射的光譜色容易做成廉價油膜感 → 錨點照對照調參，寧可收斂不要豔俗
- 玻璃 UI 的 backdrop-filter 在低階 Android 有效能坑 → 提供「關閉玻璃效果」降級開關
- 兩案並行廣告時避免自我競食 → Gemcraft 主打「動手切」、本作主打「看內在」，文案分工
